# SaudiSaaSHub

بوابة السعودية للبرمجيات كخدمة (SaaS)

##حول المشروع

SaudiSaaSHub هي منصتك الأولى لاكتشاف حلول البرمجيات كخدمة (SaaS) في المملكة العربية السعودية. قارن المنصات، اقرأ المراجعات، واعثر على الشريك التقني المناسب لاحتياجاتك.

##المميزات

- اكتشف أفضل حلول SaaS في السوق السعودي
- قارن بين مختلف المنصات والبرامج
- قراءة مراجعات المستخدمين
- البحث والتصفية حسب الفئة
- إضافة شركاتك ومنتجاتك

##التقنيات المستخدمة

- HTML5
- CSS3 (مع متغيرات CSS و Glassmorphism)
- JavaScript (vanilla)
- Arabic RTL Support

##كيفية التشغيل

1. قم بتحميل المشروع أو استنساخه
2. افتح ملف `index.html` في المتصفح
3. أو استخدم خادم ويب محلي:

```bash
# باستخدام Python
python -m http.server 8000

# أو باستخدام Node.js
npx serve
```

##النشر على Cloudflare Pages

1. ارفع المشروع إلى GitHub
2. في لوحة تحكم Cloudflare Pages، اختر "Connect to Git"
3. حدد المستودع
4. الإعدادات:
   - Build command: leave empty
   - Build output directory: /
5. انقر "Save and Deploy"
